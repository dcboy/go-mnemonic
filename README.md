# go-mnemonic

- 中文说明请见: [README.zh-CN.md](README.zh-CN.md)
- English README: [README.en.md](README.en.md)

This repository provides a deterministic converter from Chinese sentences to 24-word BIP39 mnemonics, and prints equivalent English words from the same entropy. It strictly follows BIP39 and uses the official Simplified Chinese wordlist.
